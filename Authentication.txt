Project Overview

MERN stack (MongoDB, Express.js, React.js, Node.js)
PWA features: Service Worker API, Web App Manifest, Webpack
Authentication: JWT (JSON Web Tokens) with 2FA (Two-Factor Authentication) via text message using Twilio
Frontend: React.js with Vite, Tailwind CSS, and DaisyUI

Workflow

User signs up:
User submits registration form
Server generates JWT token and sends verification code via Twilio to user's phone number
User verifies code, and server activates account
User logs in:
User submits login form
Server verifies credentials and generates JWT token
Server sends verification code via Twilio to user's phone number
User verifies code, and server returns JWT token for authenticated access
User logs out:
User requests logout
Server invalidates JWT token
